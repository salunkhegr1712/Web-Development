We used the libraries like SpeechRecognition, ecapturetwilio,wikipedia,

  1. Taking voice commands
  2. Convert voice command to computer command
  3. Run computer Computer command

Y-trees are one data Structureon which do Research activity.
This Algorithms is very specifically optimized for the search
Operations. It is basically an acyclic graph where each root
node of a level can explore min and max 3 nodes such that
It has at least one special node O called origin at the zero-level
of tree pointing to the next level nodes. The other elements in Y-tree
(if any) are sorted in a clockwise sense at each level.
Time complexity for Operations achived by this Algorithms :
  1. Insert : O(log(n))
  2. Traversal : O(n)
  3. Search : O(n/4)
  4. Find-Max : O(log(n))
